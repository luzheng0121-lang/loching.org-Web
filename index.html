01

Book & Editorial Design

Each commission begins from a structural question before a visual one — how a book opens, how it is bound, how the reader's hand moves through it. Typography, paper, and binding are treated as a single material, not three separate decisions layered on top of each other.

i.

A Practice Without a Cover

Hand-bound monograph — case study

A binding that refuses a single front: the book can be opened from either end, two halves meeting at an unmarked centre. The structure carries the argument before a single word does.

ii.

Plainsong, Reset

Poetry collection — typographic system

A single restrained grid, reused without exception across ninety pages, so that variation in the poems is never confused with variation in the design.

iii.

Field Notes, Bound

Illustrated essay collection — in progress

Hand-drawn illustration set against a deliberately quiet page — colour restraint and a structural fold used as the only joke the book allows itself.
